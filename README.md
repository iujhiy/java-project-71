This project was made to find differences in data files between two files. The application works with .json and .yaml files

## Start

```bash
make
```

## Setup

```bash
make build
```

## Run

```bash
make run
```

## Run tests

```bash
make test
```

## Run checkstyle

```bash
make lint
```

## Check update dependencies and plugins

```bash
make update
```
